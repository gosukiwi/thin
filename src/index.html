Thin

Thin is a minimal (~3kb min + gzip) modern CSS framework designed to not get in your way. It provides only the absolute base for your project, no bloat! It's mostly based on Skeleton and created using LESS and Gulp, if you are a developer and want to colaborate, please see the collaboration page in the GitHub repository wiki.

Base

Base is a reset, namely Normalize, aswell as some Skeleton-based defaults for your links, lists and images.

Typography

A base typography for your headings, paragraphs and blockquotes. All styles are in ems so if you feel like making the text bigger, remember you can make so in the container and all child elements will follow.

Grid

Grids are really simple to use! They are no more than row divs inside a container. You can use Thin's containers or create your own. What's nice about the grid is that it's responsive, go ahead, resize the browser and see how the grid nicely stacks!

Buttons

There's a button class which you can use to display nice and gently-rounded buttons. This applies automatically to all button elements aswell.

To change the color simply add a new class to your element and set the background-color as desired.

Forms

Thin also provides some base styles for your forms.

Support

Thin has been tested in the following browsers:

  • Latest Chrome
  • Latest Firefox
  • Latest Safari
  • IE 8+
  • Android 4+
  • Windows Phone IE 11

Note that some old browsers like IE 8 which don't support media queries will need some extra polyfill/conditional comment if you want to use the responsive features.

Download

The fastest and easiest way to use Thin is though a CDN, for this you can use rawgit!

                
                  <link rel="stylesheet" href="https://cdn.rawgit.com/gosukiwi/thin/master/dist/css/thin.min.css">

If you want to download the latest version, you can get it at the GitHub repo.
